### Daily driver devices

1. Dell XPS 13 9300
2. Pixel 6 Pro

### Software

1. [Nix/Nixos](https://github.com/DAlperin/dotfiles) - my laptop and cloud servers run Nixos
2. [git](https://git-scm.com/) - The best SCM with the worst UX
3. [bat](https://github.com/sharkdp/bat) - A better cat
4. [delta](https://github.com/dandavison/delta) - A better diff
5. [exa](https://github.com/ogham/exa) - A better ls
6. [ripgrep](https://github.com/BurntSushi/ripgrep) - grep but better in every way. Thanks [BurntSushi](https://github.com/BurntSushi) \<3
7. [Intellij IDEA Ultimate](https://www.jetbrains.com/idea/) - go to general purpose IDE
8. [(doom) emacs](https://github.com/DAlperin/dotfiles/tree/main/emacs) - for config files and c/c++ projects. Magit also serves as my preferred git client
9. (ocasionally) [neovim](https://neovim.io/) - anything other misc text editing needs
10. [GNOME](https://www.gnome.org/) - My DE of choice
11. [zsh](https://www.zsh.org/) - My favorite shell
12. [oh my zsh](https://ohmyz.sh/) - Zsh configuration framework
13. [terraform](https://www.terraform.io/) - Infrastructure provisioning

### Services I use

1. [1password](https://1password.com/) - My favorite password manager
2. [tailscale](https://tailscale.com/) - I really can't reccomend people try tailscale enough. It feels like magic
3. [GitHub](https://github.com/) - Where I store all my code. I also run CI/CD with GitHub workflows
4. [Purelymail](https://purelymail.com/) - Email hosting
5. [AWS](https://aws.amazon.com/) - The DNS for all of my domains is via Route53
6. [Digital Ocean](https://digitalocean.com/) - Hosting for personal projects and clients with non mission critical workloads
7. [fly.io](https://fly.io) - For highly available hosting. This site is hosted on fly geo-distributed accross the United States

### Tech stack

1. [React](https://reactjs.org) - The popular frontend framework
2. [Remix](https://remix.run) - My current favorite full stack web framework
3. [Typescript](https://www.typescriptlang.org/) - A typed language that compiles to JS. This is now mandatory for all my new JS projects
4. [Prisma](https://www.prisma.io/) - Current favorite ORM
5. [Planetscale](https://planetscale.com/) - Hosted Vitess database. It's great
6. [redis](https://redis.io) - General purpose cache and session store
7. [TailwindCSS](https://tailwindcss.com/) - A toolkit for building beautiful frontends even if you are, like me, not a designer
8. [prettier](https://prettier.io/) - Code formatting
9. [eslint](https://eslint.org/) - Enforces best practices

### Desktop Apps

1. [Brave Browser](https://brave.com/) - I begrudgingly use a chromium based browser and brave is usually the least bad one of those
2. [Spotify](https://spotify.com) - Essential for getting anything done
3. [Element](https://element.io) - Free as in beer, free as in freedom instant messaging
4. [Discord](https://discord.com) - Free as in beer instant messaging
5. [Thunderbird](https://www.thunderbird.net/en-US/) - Email client that stays out of the way and consistently works
6. For most other desktop tasks I just use the bundled GNOME apps, they do the job
